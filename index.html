SheetTrace Privacy Policy

Last updated: April 26, 2025

SheetTrace is a Chrome extension that adds Trace Precedents and Trace Dependents functionality to Google Sheets. This policy explains what data the extension accesses, how it is used, and what is never done with it.

What data SheetTrace accesses

Formula bar content (Trace Precedents — Ctrl+[): When you press Ctrl+[, SheetTrace reads the formula text currently displayed in the Google Sheets formula bar. This is done entirely within your browser. No data is sent to any server. The formula text is parsed locally to extract cell references and immediately discarded after the popup is displayed.

Spreadsheet formula data (Trace Dependents — Ctrl+]): When you press Ctrl+], SheetTrace calls the Google Sheets API v4 using your Google account credentials to fetch the formula values of all cells across all sheets in the currently open spreadsheet. This data is used solely to identify which cells contain formulas that reference your active cell. The data is held in browser memory only for the duration of that single operation and is never written to disk, sent to a third-party server, or retained after the popup closes.

Active cell address: SheetTrace temporarily stores your current cell address in browser memory to enable F5 return navigation. This is cleared when you close the tab or navigate away.

What SheetTrace does NOT do

SheetTrace does not:

  • Collect, store, or log any user data
  • Transmit any spreadsheet content to any server other than Google's own Sheets API
  • Use analytics, telemetry, or crash reporting tools
  • Track which spreadsheets you open or which cells you navigate to
  • Sell, share, or transfer any data to third parties
  • Modify your spreadsheet data in any way (read-only access only)
  • Run in the background when you are not actively using a keyboard shortcut

Google OAuth and the Sheets API

Trace Dependents requires read-only access to your Google Sheets data via the Google Sheets API v4. SheetTrace requests the spreadsheets.readonly OAuth scope, which allows it to read spreadsheet data but not modify, delete, or share it. Authentication is handled entirely through Chrome's built-in identity system (chrome.identity.getAuthToken). SheetTrace never sees or stores your Google password or credentials. The OAuth access token is used only to make the API request and is managed by Chrome.

Your use of Google services is also governed by Google's Privacy Policy.

Data retention

SheetTrace retains no data. Every piece of information accessed during a Trace Precedents or Trace Dependents operation exists only in browser memory for the duration of that operation and is not persisted anywhere.

Children's privacy

SheetTrace does not knowingly collect any information from children under the age of 13. The extension does not collect any personal information from any user of any age.
